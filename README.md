# PDV.py

# Funcionalidades Principais
Gestão de Estoque: Controle em tempo real que impede vendas acima da capacidade e alerta quando os produtos acabam.

##Checkout Financeiro: Sistema que calcula o valor total da compra baseado na quantidade solicitada.

##Validação de Pagamento: Lógica inteligente que processa o troco exato ou informa o valor exato que falta para completar a transação.

##Sanitização de Entradas: Tratamento de dados (como .lower() e .strip()) para evitar erros com espaços ou letras maiúsculas.

##Conceitos de Engenharia Aplicados
Estruturas de Repetição: Uso de loops while para manter o sistema em execução contínua.

##Lógica Condicional: Implementação de if/elif/else para tomada de decisões complexas.

##Persistência de Estado: Atualização dinâmica de variáveis para manter a integridade dos dados (estoque e saldo) durante a sessão.

##Segurança e Privacidade: Código desenvolvido com práticas de segurança, evitando o uso de dados sensíveis (hardcoding).
